# MSPR2

## Installation

```bash
$ cd backend
$ npm install
```

## Running the app

API Required a connection to a mysql database.
First copy .env.template to .env and fullfil all configuration value !

```bash
#development
$ cd backend
$ npm run start
$ cd frontend
$ live server

# product mode
$ cd backend
$ npm run start
```
